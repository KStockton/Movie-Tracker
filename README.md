
# Welcome to Movie-Tracker

## The Goals and Objectives of this project is as below:
Movie Tracker is a web application built for anyone that would like to save their favorite new releases. You can get started by creating an account. You can also see your favorites in the nav link as well as view more movie info.

## Installing / Lets Get Started
```bash
git clone [https://github.com/KStockton/Movie-Tracker] [what you want to name the repo]
```
You will also need to run the back end server, found at:

```clone the backend repo https://github.com/turingschool-examples/movie-tracker```

start backend first(local 3000) before starting frontend ui(local 3001)

Once you have cloned the repo, install the library dependencies. Run:

```bash
npm install
npm start
```
Open http://localhost:3000/
